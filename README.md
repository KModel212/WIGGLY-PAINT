# 🎨 WigglePaint

**WigglePaint** is a JavaFX-based pixel-art drawing application developed as part of the **Programming Methodology (2110215)** course.

The project demonstrates the use of **Object-Oriented Programming (OOP)** concepts through a modular drawing application with customizable tools, layered rendering, and animated export features.

---

## 📚 Course Information

**Course:** Programming Methodology (2110215)
**Institution:** Chulalongkorn University

This project focuses on applying core **OOP principles**, including:

* Encapsulation
* Abstraction
* Inheritance
* Polymorphism

These concepts are used to design a flexible architecture for drawing tools, rendering systems, and UI components.

---

## 🖌 Features

### Drawing Tools

* **Pencil Brush** – Hard pixel brush with precise 1px strokes
* **Marker Pen** – Soft edges with jitter for marker-style strokes
* **Fountain Brush** – Pressure-like effect using speed scaling
* **Highlighters** – Colorful strokes (pink, cyan, yellow)
* **Spray Brush** – Particle-style dot spraying
* **Eraser** – Removes pixels from the canvas

---

### 🎞 GIF Export

WigglePaint includes a built-in **Export to GIF** feature.

The system internally renders **two alternating frames** to produce a hand-drawn animated **wiggle effect**, creating a retro cartoon-like visual style.

---

### 🎨 Theme System

The application supports multiple visual themes using a **dynamic color palette system**.

Each theme defines:

* Foreground
* Background
* Primary
* Secondary
* Accent

When the theme changes, the UI, icons, canvas grid, and background update automatically to maintain a consistent aesthetic.

Example themes:

* Classic
* Pastel
